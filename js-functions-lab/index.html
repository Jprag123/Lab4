Part 1 · Step 1–2

Declaring & invoking greet(name)

Takes a name and returns a greeting string. Leave the box empty to see the bonus default value kick in.

— 
function greet(name = "stranger") {   // default param = bonus
  return "Hello, " + name + "!";
}

Parameters & return values addNumbers(a, b)

Accepts two parameters and returns their sum.

— 
function addNumbers(num1, num2) {
  return num1 + num2;
}

Part 3 · Step 4

Function scope changeValue()

A global x = 10 stays untouched even though a local x = 99 is declared inside the function. The local variable shadows the global — it doesn't replace it.

— 
let x = 10;            // global

function changeValue() {
  let x = 99;          // local — separate variable
  return x;
}

Part 4 · Step 5

Closures outerFunction()

The inner function remembers count from its outer function and keeps incrementing it on every click — even after outerFunction() has finished running.

count = 0 
function outerFunction() {
  let count = 0;
  return function() {
    count += 1;       // remembered between calls
    return count;
  };
}

Bonus · Another closure

Accumulator makeAccumulator()

A different closure behaviour: instead of a fixed +1 step, this one keeps a running total of every number you add.

running total = 0 
function makeAccumulator() {
  let total = 0;
  return function(amount) {
    total += amount;
    return total;
  };
}
